## 当前状态

- 当前行为：用户数据保存在本机 JSON 文件中，可通过现有导入 / 导出手动迁移。
- 当前技术形态：没有中心化账号系统、没有云同步、没有自动上传。
- 已知痛点：多设备使用需要手动导出和导入，缺少加密备份工作流。
- 现有约束：任何同步、Token、用户数据外发都必须显式 opt-in；secret gist 不能视为私有存储。
- 需要检查的文件或区域：
